# CLI Calculator

A command-line calculator built with Python to practice core programming concepts, user input handling, functions, loops, and error handling.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Division-by-zero handling
- Invalid menu choice validation
- Non-numeric input handling with `try / except`
- Continuous calculations using a `while` loop
- Exit option

## Skills Practiced

- Functions
- `if / elif / else`
- `while True`
- `break` and `continue`
- User input with `input()`
- Numeric conversion with `float()`
- Exception handling with `try / except`
- Basic input validation
