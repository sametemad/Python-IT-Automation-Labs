# Directory Inventory Reporter

A Python-based IT automation script that scans a directory and generates a structured inventory report about the files it contains.

## Features

- Scans files in a target directory
- Displays file names and sizes
- Detects file categories based on file extensions
- Counts the total number of files
- Calculates the total size of all files
- Identifies the largest file in the directory

- Conditional statements
- Loops
- Functions
- File system operations
- Working with the `os` module
- Basic data aggregation and reporting
